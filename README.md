## 🚀 Getting Started

### **Prerequisites**
- **Node.js** (v16.0.0 or higher) - [Download here](https://nodejs.org/)
- **npm** (v7.0.0 or higher) or **yarn** (v1.22.0 or higher)
- **Git** - [Download here](https://git-scm.com/)

### **Installation Steps**

1. **Clone the Repository**
   ```bash
   git clone https://github.com/sathwik-70/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in Browser**
   ```
   http://localhost:5173
   ```

### **Available Scripts**

- **`npm run dev`** - Start development server with hot reload
- **`npm run build`** - Build production-ready application
- **`npm run preview`** - Preview production build locally
- **`npm run lint`** - Run ESLint for code quality checks

### **Build for Production**

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory, ready for deployment.

### **Preview Production Build**

```bash
npm run preview
```

This serves the production build locally for testing before deployment.

## 🎨 Design Philosophy

### **Computer Hardware Aesthetic**
The portfolio embraces a cutting-edge computer hardware aesthetic that creates an immersive technological experience:

#### **Visual Elements**
- **Circuit Board Patterns**: Animated background grids resembling motherboard traces
- **Hardware Components**: Floating 3D icons representing CPU, RAM, HDD, Monitor, WiFi, and Battery
- **System Monitoring**: Real-time style displays showing CPU, RAM, and GPU usage
- **Neural Processing**: AI inference visualization with animated activity bars

#### **Color Palette**
- **Primary Gradients**: Cyan (#06b6d4) to Blue (#3b82f6) to Purple (#8b5cf6)
- **Secondary Gradients**: Purple (#8b5cf6) to Pink (#ec4899) to Rose (#f43f5e)
- **Accent Colors**: Emerald (#10b981), Orange (#f97316), Yellow (#eab308)

#### **Typography**
- **Headings**: Black font weight (900) with gradient text effects

- **Line Heights**: 150% for body text, 120% for headings

#### **Animation Principles**
- **Hardware Float**: 3D floating effects for components
- **Circuit Flow**: Moving background patterns
- **Data Transmission**: Flowing stream animations
- **Thermal Effects**: CPU heat-style pulsing

### **User Experience Design**
- **Progressive Disclosure**: Information revealed as users scroll
- **Micro-interactions**: Detailed hover and focus states
- **Visual Hierarchy**: Clear content organization and flow
- **Accessibility**: Proper contrast ratios and keyboard navigation
- **Performance**: Optimized animations and loading states

## 🔧 Customization Guide

### **Color Scheme Customization**
To modify the color palette, update the gradient classes in components:

```typescript
// Example: Changing primary gradient
className="bg-gradient-to-r from-cyan-500 to-blue-600"
// Change to:
className="bg-gradient-to-r from-green-500 to-emerald-600"
```

### **Animation Customization**
Modify animation durations and effects in the style blocks:

```css
/* Example: Adjusting animation speed */
.animate-hardware-float { 
  animation: hardware-float 10s ease-in-out infinite; 
}
/* Change to faster animation: */
.animate-hardware-float { 
  animation: hardware-float 5s ease-in-out infinite; 
}
```

### **Content Updates**

#### **Personal Information**
Update personal details in respective components:
- **Hero.tsx**: Name, title, and description
- **About.tsx**: Personal journey and stats
- **Contact.tsx**: Contact information and social links

#### **Projects**
Add or modify projects in **Projects.tsx**:

```typescript
const projects = [
  {
    icon: <YourIcon size={28} />,
    name: 'Your Project Name',
    description: 'Project description...',
    tech: ['Tech1', 'Tech2', 'Tech3'],
    status: 'Completed',
    github: 'https://github.com/username/repo',
    live: 'https://your-project.com',
    color: 'from-blue-500 to-purple-500',
    glowColor: 'blue',
    isUpcoming: false,
    progress: 100
  }
];
```

#### **Skills**
Update skill categories in **Skills.tsx**:

```typescript
const skillCategories = [
  {
    icon: <Code size={28} />,
    title: 'Your Category',
    color: 'from-blue-500 to-blue-600',
    glowColor: 'blue',
    skills: ['Skill1', 'Skill2', 'Skill3'],
    bgPattern: 'pattern-name'
  }
];
```

### **Adding New Sections**
1. Create a new component in `src/components/`
2. Import and add to `App.tsx`
3. Add navigation link in `Navigation.tsx`
4. Update the `navItems` array with the new section

### **Styling Customization**
- **Fonts**: Update font families in `index.css`
- **Spacing**: Modify Tailwind spacing classes
- **Shadows**: Adjust shadow effects for depth
- **Borders**: Customize border styles and colors
